# 台灣股票期貨 合約規格與保證金試算

提供客戶查詢台灣股票期貨（Single Stock Futures）合約規格與保證金試算的靜態網站。資料整理自臺灣期貨交易所（TAIFEX）公開資訊。

## 功能
- **合約規格一覽表**：291 檔股票期貨標的，支援關鍵字搜尋、依契約規模／保證金級距篩選、欄位排序。
- **保證金試算器**：選擇標的、輸入成交價與口數，即時算出契約價值、原始／維持／結算保證金與漲跌停價值變動。
- **共同契約規格**：到期月份、最後交易日、每日漲跌幅、最小升降單位、交易時間、交割方式等。

## 保證金計算方式
台灣股票期貨採比例保證金制度：

    保證金 = 標的成交價 × 契約乘數 × 保證金適用比例 × 口數

- 標準型契約乘數每口 2,000 股，小型（高價股）每口 100 股。
- 保證金適用比例依標的波動度分級，由期交所定期調整。

| 級距 | 原始保證金 | 維持保證金 | 結算保證金 |
|------|-----------|-----------|-----------|
| 級距 1 | 13.50% | 10.35% | 10.00% |
| 級距 2 | 16.20% | 12.42% | 12.00% |
| 級距 3 | 20.25% | 15.53% | 15.00% |
| 特殊級距 | 16%~30% | 依公告 | 依公告 |
